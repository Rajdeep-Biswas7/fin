# FinPilot AI — Personal Finance Assistant

An AI-powered personal finance platform built with Next.js, FastAPI, PostgreSQL, LangGraph, and OpenAI.

## Quick Start

See the full setup guide below or run:
```bash
docker-compose up --build
```

## Tech Stack
- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, ShadCN UI, Recharts
- **Backend**: FastAPI, Python, SQLAlchemy, JWT Auth
- **Database**: PostgreSQL, Redis
- **AI**: OpenAI GPT-4o, LangGraph, LangChain
- **Deploy**: Vercel (frontend), Railway (backend), Supabase (DB)
